# 🤖 HR Expert Chatbot

An AI-powered HR chatbot built with Python and LLaMA via Groq API.

## Features
- Ask any HR related question
- Powered by LLaMA 3.3 70B model
- Saves full chat history to JSON
- Built with Oracle HCM Cloud domain knowledge

## Tech Stack
- Python
- Groq API
- LLaMA 3.3 70B

## How to Run
1. Install dependencies: pip install groq
2. Add your Groq API key in chatbot.py
3. Run: python chatbot.py
